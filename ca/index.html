Daus

Aquest projecte tracta sobre un joc simple de daus, res complexe, simplement sort. L'objectiu del joc és aconseguir que els dos daus llençats sumin 7
És necessari estar registrat per tal de poder jugar. Tot i així NO és necessari i està desaconsellat utilitzar un correu electrònic real o una contrasenya segura, ja que els correus no són validats.
El projecte consta d'una API programada utilitzant PHP y el Framework Laravel (amb documentació en Swagger), així com una aplicació AJAX emprant Typescript (Javascript tipat) i React.

OTP - One Time Password

És tracta d'un dels mòduls del Boot Camp de seguretat informàtica de la Fundació 42 de Barcelona.
L'objectiu era crear un programa per tal de realitzar el càlcul de contrasenyes temporals d'un sol ús (Time-One-Time-Password) seguint l'especificació [RFC 6238].
El programa es pot utilitzar tant per CLI com per una interfície gràfica creada amb la llibreria de Python SimpleGUI.
Algunes opcions del programa són:
  • Seleccionar el tipus de funció per hashejar l'OTP entre vaires opcions.
  • Seleccionar el número de xifra per l'OTP.
  • Mostrar un codi QR amb els paràmetres de la contrasenya i possibilitar el ús d'un altre programa pel seu càlcul.
  • Configurar tots els paràmetres possibles per al QR.

Corsair - "Cracker" de claus RSA amb baixa aleatorietat/entropia

El programa Corsair és un software desenvolupat amb l'objectiu de desxifrar claus RSA quan l'entropia del servidor que les genera es baixa i comparteixen un dels números prims amb els quals es confeccionen.
El programa ha sigut escrit en C i les pèrdues de memòria han sigut comprovades amb l'ús de GCC i Valgrind.
A més a més s'han afegit altres utilitats al programa per a ser capaç d'entendre el mètode d'encriptació RSA.
Alguna d'aquestes funcions són::
  • Generació de claus RSA amb baixa entropia.
  • Generació de claus RSA segures emprant la llibreria OpenSSL
  • Creació de claus RSA emprant només les funcions de BigNum d'OpenSSL
  • Encriptació d'un text donada una clau públic RSA.
  • Desencriptació d'un text emprant una clau privada.
