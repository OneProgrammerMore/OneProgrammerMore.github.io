Tasks Web Application

One of the exercises of the third sprint from the Bootcamp PHP Full Stack Developer at IT Academy was to program a task application from an already given empty project.
The student had to understand the function and structure of the given code and use it in order to design and program a web application with the goal of being able to use it as a task list.
Both, the frontend and backend part of the project had to be written and linked using the given tools.

The project had 3 different levels, depending on the type of persistency used. Those are:

  • 1st Level - Using a Json File as persistency
  • 2nd Level - Using a MySQL database as persistency
  • 3rd Level - Using a MongoDB database as persistencey
Furthermore the coder had to apply the MVC pattern (Model-View-Controller) for the whole project.

Sports League Web

For the fourth sprint from the PHP Full Stack Web Development at IT Academy the student had to code a web application using the PHP Laravel framework containing at least 2 CRUDS (Create, Read, Update, Delete).
The given topic for the application was to design and program a web (backend and frontend) with the purpose of managing a football league. The two expected CRUDs were to manage the teams and the matches.
As freedom was given to the student in order to choose another topic for the project I chose to create a web application with the goal of managing several leagues from various sports.
The CRUDs programmed were:
  1. CRUD for Leagues selecting an available sport
  2. CRUD for Teams
  3. CRUD for Matches
  4. CRUD for Players
The Laravel technologies used in order to make this project were:
  • Laravel
    • Creation of the tables in the database using the artisan seed function.
    • Configuration of the different routes.
    • Programming of the various controllers used for the various CRUDs
  • Laravel Components
    • Programming of the different layouts and components
  • Breeze
    • User Authentication
    • User registration and mail confirmation
  • Livewire
    • Used in order to add Javascript for the updating of the leagues list page for fetching more leagues progressively (if existing)

Dices App

For the fifth and last sprint of the PHP Full Stack Development Bootcamp the goal was to create a simple REST API using the Laravel framework in order to simulate a simple dices game, whose goal was to throw 2 dices, sum up the value of those, and give a victory to the user if the sum was equal to "7".
The second and optional level of this sprint was to create an AJAX Application using Javascript that used the API and render to the client the different services.
I chose to use the React Library for this goal.
For the third and last optional level the student had to deploy the application.
It was possible to use a hosting server as Digital Ocean. Nonetheless I choose to use Docker in order to serve the different services from my computer to the teacher.
Some technologies used for this project were:
    PHP and Laravel Framework:
  • Code the API
  • Authentication (Passport) [OAuth2, JWT]
  • Authorization (Spatie) [Roles and Permissions]
    Javascript and React:
  • Client AJAX Application
    MySQL
  • Persistency
    Docker and Docker-Compose:
  • Deployment of the application
    Nginx And Apache Server:
  • Serve the API and the AJAX application

OTP Python Program

This project is one of several modules from the Cyber Security Bootcamp by Fundation 42 Barcelona.
The goal was to create a program in order to compute the Time-One-Time-Password following the standard [RFC 6238].
This project has been selected becasue it has a GUI made with SimpleGUI Python library.
It also has a Command Line Interface in order to use it.
Some features of this program are:
  • Selecting the Hash to compute the OTP between various possibilities.
  • Selecting the number of zifers for OTP.

  • Configure all possible parameter for the QR.

Corsair - RSA Key Cracker

The Corsair program is a software developed with the goal of cracking RSA keys when the randomness of these keys is low and the server generates RSA keys that share one of the two prime numbers used for the compute of the key.
The program has been written in the C programming language and the memory leaks have been checked with the use of the GCC and the Valgrind tools.
Some other features have been added to this software in order to be able to understand the RSA encryption method.
Some of these features are:
  • Generation of crackable RSA keys
  • Generation of secure RSA keys using OpenSSL library
  • Generation of secure RSA keys using only the BigNum functions of the OpenSSL library
  • Encryption of a text using a given public RSA key
  • Decryption of a text using a given private RSA key
