OTP - One Time Password

OTP handelt sich über ein Software um Time-One Time Passwords zu berechnen. Es ist ein Modul von die Cybersicherheit - Bootcamp von "Foundation 42 Barcelona"
Für dieses Projekt musste man die nächste Norme folgen: [RFC 6238].
Das Program kann über CLI oder GUI, das mit Python SimpleGUI entwickelt wurde, benutzt werden.
Manche Optionen von dem Software sind:
  • Die Hash-Funktion für die OTP auswählen.
  • Die Ziffer-menge des OTPs auswählen.
  • Zeigt einen QR-Code um die verschiedene OTP-Parameters mit anderen Software teilen zu können.
  • Einstellung aller mögliche Parameters für den QR-Code.

Corsair - Schwache Entropie RSA Schlüsse Cracker

Diese Projekt handelt sich über ein Software für das "Cracking" von RSA Schlüsse, wenn den Server, der die macht, hat schwäche Entropie und die Schlüssel eine von den zwei Primzählen teilen, die für ihre Anfertigung notwendig sind.
Das Programm ist auf C geschrieben und die Speicherlecks mit dem Nutzt von GCC und Valgrind korrigiert und gecheckt.
Andere Funktionen sind hinzugefügt um RSA-Kryptography besser zu verstehen.
Einige von diese Funktionen sind:
  • Erschaffung von RSA- Schlüsse mit schwache Entropie.
  • Erschaffung von sichern RSA- Schlüsse mit dem Nutzt von OpenSSL-Library.
  • Erschaffung von RSA-Schlüsse, nur mit der Nutzt von BigNum Funktionen von OpenSSL.
  • Chiffrierung einer Text mit eine öffentliche RSA Schlüssel.
  • Das Dechiffrieren einer Text mit eine private RSA- Schlüsse.
