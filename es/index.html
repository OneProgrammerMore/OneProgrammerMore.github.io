OTP - One Time Password

Se trata de uno de los módulos del Boot Camp de Cyber seguridad por la Fundación 42 en Barcelona.
El objetivo era crear un programa para realizar el cómputo de contraseñas temporales de un uso (Time-One-Time-Password) siguiendo la especificación [RFC 6238].
El programa puede ser utilizado tanto por CLI o por una interfaz gráfica creada con la librería de Python SimpleGUI.
Algunas opciones del programa són:
  • Seleccionar el tipo de función para hashear la OTP entre varias opciones.
  • Seleccionar el número de cifras para la OTP.
  • Mostrar un código QR con los parámetros de la contraseña y posibilitar el uso de otro programa para su cálculo.
  • Configurar todos los parámetros posibles para el QR.

Corsair - "Cracker" de claves RSA con baja aleatoriedad/entropía

El programa Corsair es un software desarrollado con el objetivo de descifrar claves RSA cuando la entropía de estas es baja y el servidor genera llaves que compartes uno de los números primos con los cuales se confeccionan.
El programa ha sido escrito en C y las perdidas de memoria han sido comprobadas con el uso de GCC y Valgrind.
También se han añadido otras utilidades en el software para ser capaz de entender el método de encriptación RSA.
Algunas de estas funciones son:
  • Generación de claves RSA con baja entropía.
  • Generación de claves RSA seguras usando la librería de OpenSSL
  • Creación de claves RSA usando solo las funciones de BigNum de OpenSSL
  • Encriptación de un texto dad una clave pública RSA.
  • Descifrado de un texto usando una clave privada.
